Cookie

Explicação em Inglês:

These are files that contain information such as the name and preferences of visitors to a website. This information is provided by each user on their first visit to the site. The server records the information in a file and stores it on the visitor's hard drive. Thus, when we return to the site, navigation will be faster, since the server will already have the configuration of the visitor's preferences.

Explicação em Português:

São arquivos que contêm informações como nome e preferências dos visitantes de um website. Esta informação é fornecida por cada usuário em sua primeira visita ao site. O servidor registra a informação num arquivo e armazena no disco rígido do visitante. Assim, ao retornarmos ao site, a navegação será mais rápida, pois o servidor já terá a configuração das preferências do visitante.

Cache

Explicação em Inglês:

When browsing the Internet browsers store the last images and pages visited in the cache, that is, they store in memory. The space that the browser's cache memory occupies on the computer's hard disk can be configured.

Explicação em Português:

Quando navegamos na Internet os browsers armazenam as últimas imagens e páginas visitadas no cache, ou seja, guardam na memória. O espaço que a memória cache do browser ocupa no disco rígido do computador pode ser configurado.
